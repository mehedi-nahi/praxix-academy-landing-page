Praxix Academy Landing Page
A simple static landing page/dashboard UI for “Praxix Academy”, built with plain HTML and CSS (no frameworks).

Features
Sidebar navigation (Account, Academy sections)
Main chat/content area mock UI
Cards for: File Upload, Lecture Sheet, Quiz Creation
Responsive layout is not guaranteed (layout is mostly fixed-width for the design)
Project Structure
index.html — main page markup
style.css — all styling for the layout and cards
Asset folders used by the UI: Sidebar/, Academy/, Account/, card/, navbar/, home-right/
How to Run
Open index.html in your browser.
Keep the project folder structure intact so image paths work correctly.
Notes
This is a front-end UI mockup. Buttons and chat inputs are not connected to a backend.
